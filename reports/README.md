# Reports Directory

This directory contains generated analysis reports and visualizations:

- `analysis_report.json` - Detailed analysis data in JSON format
- `recommendations.txt` - Text recommendations for students
- `activity_distribution.png` - Histogram of activity types
- `average_grades.png` - Bar chart of average grades by activity
- `engagement_distribution.png` - Pie chart of engagement levels

These files are generated by running `python src/main.py`
